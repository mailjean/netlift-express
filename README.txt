
# Netlift Express Demo Setup

1. Go to https://console.firebase.google.com/
2. Create a new project (e.g., netlift-express)
3. Add Firestore Database and enable authentication (email/password)
4. Get your Firebase config and replace it in firebase.js
5. Upload the site to https://app.netlify.com/ or Replit to get a free domain
6. Use 'admin.html' to log in and manage parcels

Enjoy!
